This section highlights advanced navigation and estimation coursework, focusing on GNSS data processing, filtering, and precision positioning.

Projects

Differential GNSS Baseline Estimation

Goal: Compute precise relative positioning between two receivers.

Methods: Epoch-by-epoch least squares, ENU/ECEF conversion.

Results: Quantified baseline accuracy and convergence behavior.

Kalman Filtering & Smoothing

Goal: Improve navigation accuracy over time.

Methods: Constant-velocity KF, tuning, residual analysis.

Results: Reduced noise and improved stability.

Carrier-Phase Processing & Ambiguity Resolution

Goal: Achieve high-precision relative positioning.

Methods: Double-difference code + phase, LAMBDA algorithm.

Results: Sub-meter (and better) positioning performance.

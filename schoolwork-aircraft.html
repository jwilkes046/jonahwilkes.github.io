Project

Aircraft Design Project

MATLABSimulinkXFLR5Technical Writing

The goal of this project was to develop a multi-functional agricultural aircraft, with autonomous functionality when dusting crops, as well as with space for manual piloting during ferry missions. Throughout the semester, the project progress from a requirements-driven Conceptual Design Report (mid-semester) to final aircraft model and performance evaluation in a Final Design Report. My work focused on sizing an aerodynamic decisions early, then on building an aerodynamic dataset using XFLR5 and integrating it into a flight dynamics/control simulation for maneuver performance using Simulink.

Conceptual Design Report (Mid-Semester)

Preliminary design parameters and justification of key configuration choices.

Design Approach

I documented a requirements-driven approach that connects mission needs (payload, flight profile, and operational constraints) to a feasible baseline configuration. The goal was to make early decisions traceable: every geometry or sizing choice had a performance or constraint-based reason behind it.

Initial Sizing (W/S + Weight Iteration)

I performed the first-pass sizing using wing loading as the primary driver, then iterated fuel fraction, empty weight fraction, and gross weight estimation to converge on a consistent design point. This included trade study logic to understand how sizing choices moved the aircraft through the feasible design space.

Aerodynamics (Airfoil + Wing Sizing)

I evaluated candidate airfoils using performance-relevant metrics and summarized the decision in a weighted selection matrix. The chosen airfoil and wing sizing were then carried forward as the aerodynamic baseline for the final configuration and later simulation work.

I built and refined the aircraft configuration in XFLR5 to generate aerodynamic outputs across relevant operating conditions. This established the baseline coefficient curves used later for performance and control simulation.

XFLR5 Data → MATLAB Processing

I developed MATLAB scripts to extract XFLR5 results and compute coefficient increments (e.g., changes due to control deflection or other configuration effects). These processed outputs were packaged into a form suitable for lookup/interpolation inside the simulation environment.

Data extractionCoefficient incrementsLookup tables

Simulink Control & Maneuver Evaluation

I integrated the processed aerodynamic data into a Simulink-based flight model and used it to evaluate aircraft behavior in multiple scenarios. The final analysis emphasized stability and control behavior under representative maneuvers and flight phases.

Controls integrationStabilityManeuvers

Performance Cases

  • Glide: energy management and trim behavior.
  • Climb: climb capability across operating points.
  • Hold: sustained maneuver/loiter behavior and control authority.
  • Landing: approach/landing phase constraints and handling.

GPS Positioning Testing

At the same time I was completing this course, I was also taking Introduction to Navigation Systems, which got me thinking about how GPS positioning would work for an aircraft like this. So, I developed an MATLAB code to express a short GPS positioning representation of the aircraft's final approach flight path, using satellite position estimation concepts from my concurrent Navigation coursework. The image on the right is the result of my modeling, which included the simulation of 32 satellites at different orbits, their measurements of the aircraft's position (in ECEF coordinates) at each second, random error due to sources such as receiver noise, satellite and receiver clock offsets, and ionosperic and tropospheric delays consistent with GPS error sources, and the simulated correction of this error using typical GPS filtering techniques.

While I do admit that this is a completely arbitrary and user-controlled method of estimating position, I included this in my report as a test of what I learned and understood about Navigation, as well as an exploration of the applications of GPS positioning in the Aerospace field.
